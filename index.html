The pipeline

Seven specialist agents,
working in parallel sync.

01

Strategist

Reads the repo's manifest, dependency tree, and changelog to scope the audit. Picks the right tool chain — bandit, semgrep, trufflehog, or a custom Python rule pack.

claude-opus-4.7
02

Researcher

Pulls the latest CVE database for every dependency. Cross-references with HN, GitHub Issues, and PyPI to surface known issues with similar code patterns.

exa + context7
03

Builder

Spins up a sandboxed scan environment. Pins tool versions for reproducibility. Outputs JSONL findings with file:line:rule_id and a confidence score.

docker + bandit
04

Auditor

Reviews each finding against three filters: real exploitability, blast radius, and remediation cost. Drops noise, keeps signal. This is where senior reviewer judgement lives.

claude-sonnet-4.6
05

Outreach

Drafts a pitch personalized to the maintainer's role and the finding's severity. Verifies recipient via MX + SMTP RCPT before send. No bounces, no spam flags.

gmail smtp + dns-py
06

Verifier

Watches inbox for replies, classifies sentiment, and flags qualified leads. Auto-replies with the proof-of-finding bundle when interest is real.

imap + nlp
07

Reviewer

Closes the loop. Tracks remediation status, requests follow-up audits at thirty / sixty / ninety day intervals, and writes the public retrospective.

supabase + cron
